## Назначение

Этот репозиторий является **структурным индексом фреймворков всех веток** внутри экосистемы **Lizerium**.

Он не является основной реализацией, а служит как:

* карта фреймворков относящихся к экосистеме

---

## Что входит в это направление

### 1. 🌌 Фреймворки
